# Kakaon

## 1. 프로젝트 개요

### 가맹점 운영을 실시간으로 파악하고, 매출·거래·이상거래를 한눈에 확인하는 가맹점 매출 관리 플랫폼

### 기획 배경

- **파편화된 매출 데이터**: 결제 수단이 다양해짐에 따라 데이터가 분산되어 통합 관리가 어려움

- **실시간성 부족**: 기존 ERP나 POS 시스템에 의존하여 실시간 매출 현황 파악의 한계 존재

- **이상 거래 대응 한계**: 갑작스러운 취소율 급증이나 고액 결제 등 이상 징후에 대한 즉각적 대처 불가능

- **실질적 인사이트 부재**: 단순 데이터 나열을 넘어 실제 운영 개선으로 이어질 분석 정보 부족

### KAKAON의 해결책

- **통합된 데이터 + 실시간 모니터링 + 직관적 시각화 + 운영 인사이트 제공**
- 흩어진 매출 데이터를 하나로 모으고, 실시간으로 이상 징후를 탐지하며,
  사장님이 직관적으로 이해할 수 있는 분석 리포트를 제공하여 **매장 운영의 효율성을 극대화**했습니다.

### 주요 특징

1. **실시간 대시보드** – 오늘 매출, 이달 매출, 전일 대비 변화율, 이상거래 건수 즉시 확인
2. **지능형 매출 분석** – 기간·요일·시간대별 매출 패턴 분석, 결제수단 비중 분석
3. **이상거래 탐지 시스템** – 취소율 급증, 위치 기반 결제 위험, 반복 결제 등 자동 감지
4. **거래 내역 통합 관리** – CSV 업로드·다운로드, 승인번호 검색, 상세 주문 정보 조회
5. **가맹점 관리** – 매장 정보·운영 상태·알림 수신자 관리
6. **메뉴 및 주문 관리** – 매장 메뉴 CRUD, 주문 내역 조회 및 취소
