# Kakaon

## 1. 프로젝트 개요

### 가맹점 운영을 실시간으로 파악하고, 매출·거래·이상거래를 한눈에 확인하는 가맹점 매출 관리 플랫폼

### 기획 배경

### 주요 특징 
1. **실시간 대시보드** – 오늘 매출, 이달 매출, 전일 대비 변화율, 이상거래 건수 즉시 확인
2. **지능형 매출 분석** – 기간·요일·시간대별 매출 패턴 분석, 결제수단 비중 분석
3. **이상거래 탐지 시스템** – 취소율 급증, 위치 기반 결제 위험, 반복 결제 등 자동 감지
4. **거래 내역 통합 관리** – CSV 업로드·다운로드, 승인번호 검색, 상세 주문 정보 조회
5. **가맹점 관리** – 매장 정보·운영 상태·알림 수신자 관리
6. **메뉴 및 주문 관리** – 매장 메뉴 CRUD, 주문 내역 조회 및 취소

## 3. 주요 기능 소개

### 1) OAuth 회원가입 / 로그인

![회원가입](docs/gif/회원가입.gif)

- Kakao OAuth 기반 간편 회원가입/로그인

### 2) 대시보드 : 오늘·이번달 매출 현황

![대시보드](/docs/images/대시보드.gif)

- 오늘 기준 실시간 총 매출 
- 전일 대비 증감률 및 추이 그래프
- 이달 매출 일자별 그래프
- 오늘 발생한 이상거래 건수 실시간 알림

### 3) 매출 분석 : 기간별 상세 통계

- 기간별 조회 : 이번주 / 이번 달 / 올해 / 지정 기간 매출
![매출 분석](/docs/images/기간별매출.gif)

- 패턴 분석 : 시간대별 평균 매출 , 요일별 매출 추이
![매출 분석](/docs/images/시간대별매출.gif)

- 결제 수단 : 카드, 현금, 앱 결제 등 수단별 비중 파이 차트 제공
![매출 분석](/docs/images/결제수단별비중.gif)

- 가맹점 별 매출 : 보유한 가맹점 간 매출 비교 분석
![매출 분석](/docs/images/가맹점별매출.gif)

- 취소율 추이 : 월별 취소율 추이 그래프 제공
![매출 분석](/docs/images/취소율추이.gif)

- AI 매출 분석 리포트 : 매출 데이터를 기반으로 한 AI 인사이트 및 요약 리포트 제공
![매출 분석](/docs/images/ai매출리포트.gif)

### 4) 거래 내역 관리

![거래 내역](/docs/images/거래내역.gif)

- 필터링 : 기간, 결제수단, 결제상태(승인/취소), 배달여부 필터 적용
- 빠른 검색 : 승인번호, 주문번호 검색 지원
- 데이터 관리 : 대용량 거래 내역 CSV 업로드 및 엑셀 다운로드 지원

### 5) 이상거래 탐지

![이상 거래 탐지](/docs/images/이상거래고액결제.gif)

- 실시간 모니터링 : 이상 징후 발생 시 즉시 알림 전송
- 다양한 탐지 유형 : 
  - 취소율 급증 감지 (전주 대비)
  - 동일 결제수단의 물리적 이동 불가능 거리 결제
  - 고액 결제 급증 및 영업시간 외 결제
- 알림 : 미확인 알림 리스트 조회 및 확인 처리
